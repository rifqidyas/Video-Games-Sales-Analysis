#  Video Game Sales Analysis

##  Project Overview
This project analyzes global video game sales data to identify:
- Best-selling platforms
- Platform lifecycle trends
- Genre performance
- Regional sales differences

The goal of this project is to practice data cleaning, exploratory data analysis (EDA), and data visualization using Python.

---

##  Dataset
Dataset: Video Game Sales  
Features include:
- Name
- Platform
- Year
- Genre
- Publisher
- NA_Sales
- EU_Sales
- JP_Sales
- Other_Sales
- Global_Sales

---

##  Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

##  Project Workflow

### 1️ Data Cleaning
- Handle missing values
- Convert data types
- Remove duplicates
- Validate year range

Notebook: `01_data_cleaning.ipynb`

---

### 2️ Exploratory Data Analysis (EDA)
- Global sales by platform
- Platform lifecycle (min & max year)
- Sales distribution
- Platform comparison

Notebook: `02_eda_and_visualization.ipynb`

---

##  Key Findings (Current Progress)

- PS2 shows the highest global sales.
- Several platforms such as X360, PS3, Wii, and DS show medium-level sales performance.
- Platform lifecycle duration does not always correlate with highest sales.

(Analysis still in progress.)
